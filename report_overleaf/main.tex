\begin{abstract}
This report documents my full competition path from an initial pure-ML approach to a structure-aware solution based on leakage reconstruction and tail-only forecasting. I submitted over 200 experiments and used that iteration budget to test hypotheses, not only hyperparameters. The final pipeline reconstructs targets deterministically where future \texttt{feature\_16} values are observable, then applies fallback strategies only on the terminal unknown windows (last 10/60/240 rows for short/medium/long). I provide code-level proof for every core claim: dataset structure, leakage equations, test coverage, unknown-tail localization, boundary simulation, and submission behavior comparison.
\end{abstract}
